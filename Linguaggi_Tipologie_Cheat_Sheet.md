# Cheat Sheet Tipologie Linguaggi di Programmazione

| Categoria | Descrizione / Esempi |
|----------|------------------------|
| **Tipizzazione** | **Statico:** C, C++, Java, Ada, Rust, C#, Swift  <br> **Dinamico:** Python, JavaScript, Lisp, R  <br> **Forte:** Python, Java, Haskell, Scala, Swift  <br> **Debole:** JavaScript, PHP, Forth  <br> **Opzionale:** Python (type hints), TypeScript, Julia |
| **Paradigmi** | **Procedurale:** C, Pascal, Fortran  <br> **Funzionale:** Haskell, Lisp, F#, Scala, OCaml, Elixir  <br> **OO:** Java, C++, Python, C#, Swift, Kotlin  <br> **Logico:** Prolog  <br> **Imperativo:** C, Go, Fortran, JavaScript  <br> **Dichiarativo:** SQL, HTML, CSS  <br> **Event-driven:** JavaScript, C#, Swift  <br> **Multiparadigma:** Python, Scala, Kotlin, Julia |
| **Funzioni / Polimorfismo** | **First-class functions:** Lisp, Python, JS, Haskell  <br> **Polimorfismo parametrico:** Haskell, Scala, OCaml  <br> **Generics:** Java, C#, Rust, Go |
| **Esecuzione / Memoria** | **Compilato:** C, C++, Rust, Go, Swift  <br> **Interpretato:** Python, JS, Lisp, R, Bash  <br> **JIT:** Java, C#, Julia  <br> **Garbage collected:** Java, Python, Go, C#  <br> **Manuale:** C, C++ |
| **Avanzate / Stile** | **Side effects:** C (sì), Python (sì), Haskell (controllati via monadi)  <br> **Lazy eval:** Haskell (di default), Python (solo generatori)  <br> **Dispatch:** C++ (static + virtual), Java (dynamic)  <br> **Mutable/Immutable:** Python (entrambi), Haskell (immutabile)  <br> **Metaprogrammazione:** Lisp (macro), Rust (macro), C++ (templates), Python |
| **Livello / Scopo** | **Low-level:** C, Assembly, Rust  <br> **High-level:** Python, Java, R, Julia  <br> **Domain-specific:** SQL, MATLAB, Verilog  <br> **General-purpose:** Python, C++, Go, Java |
| **Concorrenza** | **Thread/processi:** Go, Erlang, Java, Ada, Scala, Rust  <br> **Actor model:** Erlang, Akka (Scala) |
| **Gestione Errori** | **Eccezioni:** Java, Python, C#  <br> **Tipi sicuri:** Rust (Result/Option), Haskell (Maybe/Either)  <br> **Valori di ritorno:** Go, C |
| **Ecosistemi / Tooling** | **Package manager:** pip, npm, cargo, maven, nuget, go modules  <br> **Ambienti:** JVM (Java/Scala), CLR (.NET), Swift toolchain, Julia REPL |
