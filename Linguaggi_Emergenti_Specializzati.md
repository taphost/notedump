# Cheat Sheet Linguaggi Emergenti e Specializzati (Esteso)

| Linguaggio | Categoria / Ambito | Caratteristiche principali | Paradigma / Tipizzazione |
|------------|--------------------|-----------------------------|--------------------------|
| **Zig** | Sistemi, C-replacement | Sicuro senza GC, controllo memoria, build integrato | Imperativo, statico |
| **Nim** | Sistemi, scripting | Sintassi Python-like, compilato, veloce | Multiparadigma, statico |
| **Crystal** | Backend moderno | Ruby-like, compilato, inferenza forte | OO, statico |
| **Dart** | Mobile/web (Flutter) | AOT+JIT, cross-platform | OO, statico |
| **TypeScript** | Web | JS tipizzato, tooling eccellente | OO/funz., statico opz. |
| **Lua** | Scripting, giochi, embedded | Leggero, integrabile, veloce | Imperativo, dinamico |
| **Elixir** | Backend distribuito | BEAM VM, concorrenza massiva | Funzionale, dinamico |
| **Rust (avanz.)** | Sistemi sicuri | Ownership, zero-cost abstractions | Multiparadigma, statico |
| **Go (avanz.)** | Cloud-native | Goroutine, semplicità estrema | Procedurale, statico |
| **Julia** | Scientifico/AI | Performance numeriche altissime | Funz./imper., dinamico |
| **GraphQL** | Query API | Schema-driven, query strutturate | Dichiarativo |
| **Cypher** | Database a grafo | Query relazionali su grafi | Dichiarativo |
| **Svelte** | Web frontend | Compila in JS ottimizzato | Dichiarativo, reattivo |
| **Elm** | Web funzionale | Zero runtime errors | Funzionale puro |
| **Red** | Embedded/UI | Sintassi minimale | Multiparadigma, dinamico |
| **V** | Compilato leggero | Compilazione istantanea, no runtime | Imperativo, statico |
| **Kotlin (avanz.)** | JVM/Android | DSL potenti, sicurezza null | OO/funzionale, statico |
| **Deno** | Runtime JS/TS | Sicuro, TS nativo | Event-driven |
| **Haxe** | Cross-platform | Compila in molti target | Multiparadigma, statico |
| **Ballerina** | Cloud/Microservices | API-first, integrazione cloud | Statico |
| **Futhark** | GPU/parallelismo | Ottimizzato per GPU, funzionale puro | Funzionale |
| **Gleam** | BEAM moderno | Funzionale statico su Erlang VM | Statico |
| **Bosque** | Linguaggi regolarizzati | Elimina mutabilità implicita | Statico |
| **Mojo** | AI/acceleratori | Superset veloce di Python | Statico/dinamico |
| **Carbon** | Post-C++ | Interoperabile con C++ | Statico |
| **Wren** | Scripting moderno | Embedded, leggero | Dinamico |
| **Janet** | Lisp per scripting | Piccolo, veloce | Dinamico |
| **OCaml (moderno)** | Sistemi, compilatori | Pattern matching potente | Statico |
| **Idris2** | Tipi dipendenti | Linguaggio formale | Statico avanzato |
| **Roc** | Funzionale semplice | No runtime errors | Funzionale |
